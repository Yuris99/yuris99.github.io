The Traveling Salesperson Problem

  • Problem Definition

    • Determine a shortest route that starts at the salespersonâ€™s home city, visits each of the cities once, and ends up at the home city

  • Adjacent matrix W

    • V = set of all the vertices
    • A = a subset of V
    • D[vi][A] = length of a shortest path from vi to v1
              passing through each vertex in A exactly once

  • Length of an optimal tour = minimum(W[1][j] + D[vj][V - {v1, vj}])
